낮에는 일하며 프로그램 하고  
밤에는 미술학원 다니며 그림그리면 얼마나 좋을까.   
  
대학원이 리듬을 깨고 있어 OTL   

utility callback\_test " box rotate"  
(  
 slider left\_box "Left" range:[-90,90,0] orient:#vertical offset:[-0,0]  
 slider right\_box "Right" range:[90,-90,0] orient:#vertical offset:[0,-0]  
 button resetv "Reset" width:140  
   
\* 인터페이스 만들기  
  
on resetv pressed do  
(  
 $Rightbox.rotation.x\_rotation = $Leftbox.rotation.x\_rotation = 0  
 left\_box.value = right\_box.value = 0  
)  
  
\* 리셋버튼 눌렀을때의 동작

on left\_box changed val do   
(  
 $Leftbox.rotation.x\_rotation = -val  
)  
  
\*레프트박스의 값 변화에 따른 동작

on Right\_box changed val do   
(  
 $Rightbox.rotation.x\_rotation = -val  
)  
  
\* 라이트박스의 변화에 따른 동작.

)
