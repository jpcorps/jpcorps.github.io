하고 싶은 거 정리 ㅋ

* 캐릭터

+ Bone을 이용한 커스터마이드 - face와 body 커스터마이즈.
  - face는 입 움직임과 표정 움직임이 가능하도록 고려.(연출시 필요)
+ 캐릭터 IK 적용
+ base texture 2장 (일반/문신) / normal map / glossmap / illumination(림 라이트 효과) / 염색 채널

* 라이트

+ GI 렌더링 툴 제작  및 맵에디터 내장
  - 글로벌 일루미네이션 쉐도우맵 자동생성.
+ 다이나믹 라이트 삽입 기능
  - 맵에디터에서 삽입. attenuation 과 색상, 강도 조절 가능
+ ambient cube 적용
  - 지역별 환경에 따라 변화할 수 있게 처리

* 일반

+ 퀘스트 연출 제작 툴
+ 연출을 위한 루아 스크립트 적용 (퀘스트 연출과 전투 연출 카메라 조작을 위함)
+ 지글본 레벨의 물리효과 적용.
+ 트리거 제작을 위한 오브젝트 클래스 제작 / 맵에디터 적용.
  - 영역 지정 기능. 트리거 발생 시 루아 스크립트를 통한 조작 가능
+ HDR 효과

* 쉐이더

+ 쉐이더의 용이한 추가/삭제를 위한 구조

* 배경
* + 쿼드트리를 통한 오클루젼 컬링 / 클리핑 기능
  + 원거리 스카이라인을 위한 단색 텍스쳐 원경
    - fog 칼라와 연동된 2\*2 텍스쳐 생성
  + 물 효과 - 프레넬 효과, 굴절, 실시간 반사, 물 외각 부분의 물결 표현
  + 타일의 사이즈 조작 기능
    - 타일의 알파와 텍스쳐를 분리하여, 알파는 그대로 둔 채 텍스쳐의 타일 사이즈만 조작 가능
* 이펙트
  + 전용 파티클 이펙트 툴 제작
* LOD
  + polygon LOD
  + Bone LOD
  + Shader LOD
  + Terrain LOD
* 사운드
  + 지역 사운드 효과를 위한 작업 / 맵에디터 개선
* UI
  + 스케일폼 엔진 지원
* 최적화

+ 최적화 작업을 위한 퍼포먼스 체크 툴 제작
  - 클라이언트에 로그 파일 생성하여 추후 이 파일을 홈페이지 버그 게시판 작성 시 자동전송 가능
