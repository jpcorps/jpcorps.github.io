알 수 있는 정보 : 그래픽 카드 이름 / 그래픽 카드 메모리 양 / 쉐이더 지원여부 / CPU 코어수 / 물리 메모리   
  
위 내용중 간략한 정보를 읽어내 자동으로 처리합니다. 물론 오차가 있을 수 있으므로 (인텔 칩 그래픽 카드 같은 경우)   
여기에 추후 추가되는 if 문이 있을 것으로 예상할 수 있습니다. 그건 테스트하면서 추가할 수 밖에 없겠네요.. 변수가 많아서.   
  
  
쉐이더 지원여부 : 최소 2.0 지원 / 3.0 지원 권장  
  
그래픽 카드 메모리 양 : 128~ 256M미만 1점       256M~ 512M미만 2점        512M 이상  3점  
CPU 코어 :                  1개일때 1점                   2개일때 2점            3개 이상으로 인식될때 3점  
물리 메모리 :                1G 혹은 이하 일때 1점                  2G 이상일때 2점      3G 이상일때 3점   
  
  
모든 점수를 합산했을때   
최상옵션 : (쉐이더 3.0 지원) && (그래픽 카드 메모리 3점) && (CPU와 물리 메모리 5점 이상 )  
중간옵션 : (쉐이더 3.0 지원) && (그래픽 카드 메모리 2점) && (CPU와 물리 메모리 3점 이상 5점 미만 )  
최하옵션 : (쉐이더 2.0 지원) && (그래픽 카드 메모리 2점 || 1점) && (CPU와 물리 메모리 3점 || 그 이하 )
