| 게임서 난폭운전, 길에서도 난폭해진다? |
|  |
| [조선일보 2007-03-30 03:13] |
| |  | | --- | |  |   |  | | --- | |  |    폭력적인 게임을 즐기면 실제 성격이나 행동도 폭력적으로 변할까. 게임을 둘러싼 오래된 논쟁에 다시 불이 붙었다. 일본 세가(Sega)의 레이싱 게임 ‘크레이지 택시: 요금 전쟁’이 이같은 논란의 주인공이다.  이 게임은 휴대용 게임기 플레이스테이션 포터블(PSP)로 즐긴다. 택시 운전사가 된 게이머가 수단·방법을 가리지 않고 손님을 목적지까지 얼마나 빠르게 데려다 줄 수 있는가를 겨루는 게임이다. 정해진 시간 안에 손님을 데려다 주면 플레이 시간이 늘어나면서 계속 게임을 진행할 수 있다.   문제는 이 게임에서 좋은 점수를 따려면 아주 난폭하고 공격적으로 택시를 몰아야 한다는 것이다. 운전자가 손님을 원하는 목적지까지 데려다 주고 더 많은 돈을 벌기 위해서는 과속은 기본이다. 사람들이 오가는 인도(人道)로 돌진하거나 온갖 장애물을 택시로 들이받기도 한다.   세가조차 “아주 숙련된 운전자라도 이 게임을 제대로 하려면 별도의 적응이 필요하다”고 밝히고 있다.  이와 관련, 독일 연구팀은 레이싱 게임이 실제 운전습관에 어떤 영향을 미치는지 실험을 실시했다. 연구팀은 국제 학술지 ‘실험 심리학 저널’ 최신호에서 “레이싱 게임을 자주 하는 사람일수록 진짜 도로에서 차를 몰 때도 경쟁적이거나 무감각한 운전 태도를 보이며, 자동차 사고횟수와도 비례 관계가 있다”는 내용의 보고서를 발표했다.  연구자들은 “레이싱 게임을 한 이후에 운전석에 앉은 실험자들은 평상시보다 훨씬 더 흥분했으며, 위험한 도로교통 상황에서도 안전운전보다 과감한 운전을 감수하려 했다”고 밝혔다. 또 게임을 즐기던 남성 운전자들은 위험한 도로 상황에서 게임을 하지 않은 사람보다 1초 가량 늦게 반응하는 것으로 나타났다. 흥미롭게도 남성 게이머가 여성보다 게임의 영향을 훨씬 더 많이 받는다는 사실도 연구팀은 발견했다.  이와 관련, 미국 하버드대학 의대 교수이자 보스턴 아동병원 의사인 데이비드 비크햄 박사는 “레이싱 게임과 운전 기술의 상관관계가 요즘처럼 주목받은 적이 없다”며 “비디오 게임이 운전 행태에 변화를 가져올 수 있다는 연구결과는 놀라운 사실이 아니다”라고 말했다. 그는 “전투기 조종사들이 비행게임으로 훈련하고, 육군도 군인을 훈련시키기 위해 비디오 게임을 사용한다”고 설명했다. 비디오 게임이 특정한 기술을 가르치는데 활용될 수 있다는 말이다.  폭력성이 강한 레이싱 게임을 즐기는 청소년이 실제로 거리에서 보행자를 칠 것이라고 속단하기는 어렵다. 하지만 적어도 이들이 일반인보다 위험하거나 나쁜 운전자가 될 가능성이 높은 것은 사실이라고 독일 연구팀은 지적했다.  비크햄 박사는 “독일 연구팀의 보고서에 담긴 진정한 메시지는 부모가 자신의 아이들이 무슨 게임을 사서 즐기는지 관심을 갖고 지켜봐야 한다는 것”이라며 다음과 같이 권고했다.  “아이들은 자신이 인지하는 모든 것을 통해 세상을 배웁니다. 부모는 그 교육의 1차적 원천이 돼야 합니다.”   [탁상훈 기자 [if@chosun.com](mailto:if@chosun.com)] <모바일로 보는 조선일보 속보 305+NATE, 305+magicⓝ([http://mobile.chosun.com](http://mobile.chosun.com/))> - Copyrights ⓒ 조선일보 & chosun.com, 무단 전재 및 재배포 금지 |
